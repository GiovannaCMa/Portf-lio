Projetos

Web Developement

O "Poemas de Gigi" é um site pessoal que desenvolvi para compartilhar minha paixão pela escrita. O projeto apresenta uma coleção de poemas autorais, com destaque para o livro "Poemas de Gigi - Os Amores", que explora sentimentos intensos de amor, paixão e amizades transformadas em algo mais. Além disso, o site oferece aos visitantes a oportunidade de solicitar poemas personalizados, enviando suas histórias por meio de um formulário de contato.

Back-End Developement

O projeto é um controlador REST criado com o framework Spring Boot, projetado para lidar com operações de upload, download e comparação de imagens. Ele expõe endpoints para enviar imagens, buscar imagens existentes e deletar imagens pelo ID. Além disso, contém uma funcionalidade para comparar uma imagem carregada e retornar a mais similar da base de dados.
